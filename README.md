# Stock Broker Client Dashboard Application

This project is a **Stock Broker Client Dashboard Application** developed as part of an placement assignment.

The application allows users to **sign up, log in, view available stocks, and manage a personal watchlist**.

## Project Overview

The main goal of this project is to understand how a basic stock broker dashboard works using a full-stack approach.
Users can subscribe to stocks they are interested in and manage them through a personal dashboard.

## Application Pages

* **Sign Up Page**
  Users can create a new account using name, email, and password.

* **Login Page**
  Existing users can log in using their credentials.

* **Dashboard Page**

  * Welcome message with user name
  * List of **Available Stocks**
  * **My Watchlist** section
  * Subscribe and unsubscribe functionality
  * Logout option

## Features

* User authentication (Sign Up & Login)
* View available stocks with prices
* Subscribe and unsubscribe stocks
* Personal watchlist per user
* Clean and simple UI

## Technologies Used

* Frontend: React (Hosted on Vercel)
* Backend: Node.js, Express (Hosted on Railway)
* Database: MySQL (Railway)
* Styling: CSS
* Version Control: Git, GitHub

## How to Run Locally

1. Clone the repository

   ```bash
   git clone https://github.com/SANKETH-CHANDRA-SHETTY/stock-broker-app.git
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start backend server

   ```bash
   npm start
   ```

4. Start frontend

   ```bash
   npm run dev
   ```
